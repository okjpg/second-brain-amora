## Como criar uma skill

1. Crie a pasta: `skills/{nome-da-skill}/SKILL.md`
2. Adicione nesta tabela
3. A skill fica disponivel como `/nome-da-skill` no Claude Code

## Organizacao (quando crescer)

Com 5+ skills, organize por categoria:

```
skills/
├── _registry.md              # Este arquivo (indice global)
├── operacoes/
│   ├── _registry.md          # Indice da categoria
│   └── minha-skill/SKILL.md
├── conteudo/
│   └── outra-skill/SKILL.md
└── ...
```
